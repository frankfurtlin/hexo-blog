# 海报生成器

## 1. 安装 docker

宝塔面板安装 docker

## 2. 拉取 fastposter 镜像

[fastposter主页](https://poster.prodapi.cn/doc/)

``` bash
# 首先在服务器安全组放开5000端口
$ docker run -it -e POSTER_URI_PREFIX=http://ip:5000/ --name fast-poster -p 5000:5000 tangweixin/fast-poster
```
